```uml
@startuml
[*] --> トップページ


トップページ -down-> 記事
トップページ -> アカウント
トップページ -> 投稿画面 : 投稿をクリック

トップページ:・記事一覧

トップページ -> ログイン画面 : ログインをクリック


state ログイン画面{
ログイン -> 新規登録
ログイン -down-> ログイン完了 : ログイン完了
新規登録 -down-> 登録完了 : 登録完了
}

state 記事{
記事一覧 -left-> 記事詳細 : 記事をクリック
記事詳細 -> ログイン画面 :コメント・スタンプ(未ログイン)

記事一覧:・ランダムで表示
記事一覧:・新着順で表示
記事一覧:・カテゴリ別で表示
記事一覧:・観覧数ランキングで表示
記事一覧:・お気に入り登録者数で表示
記事一覧:・特定アカウントで表示

記事詳細:・記事をお気に入りに追加
記事詳細:・コメント
記事詳細:・スタンプ
記事詳細:・記事の報告・通報
記事詳細:・記事読了後にランダムで表示

}

state 投稿画面{
投稿画面 -> ログイン画面 : 未ログイン
投稿画面 -> 投稿 : ログイン済み

投稿:・記事を投稿
投稿:・動画を投稿
投稿:・時間指定して投稿
投稿:・記事の公開非公開の設定
投稿:・サムネイルを設定

}

アカウント -> 記事詳細 : 記事をクリック

アカウント:・プロフィール（紹介文など）
アカウント:・アカウント編集
アカウント:・投稿記事
アカウント:・観覧記事の履歴一覧表示(編集、削除)
アカウント:・お気に入り記事一覧表示
アカウント:・お気に入り登録全アカウントの投稿記事一覧表示
アカウント:・お気に入り登録アカウント一覧表示


[*] -right-> 管理者
管理者:・記事削除
管理者:・ユーザー削除


@enduml
```
