```uml
@startuml
[*] --> トップページ

state 記事{
記事一覧 -> 記事詳細 : 記事をクリック
特殊記事一覧 -> 記事詳細 : 記事をクリック

記事一覧:・ランダムで表示
記事一覧:・新着順で表示
記事一覧:・カテゴリ別で表示
記事一覧:・観覧数ランキングで表示
記事一覧:・お気に入り登録者数で表示
記事一覧:・特定アカウントで表示

特殊記事一覧:・観覧記事の履歴一覧表示
特殊記事一覧:・お気に入り記事一覧表示
特殊記事一覧:・お気に入り登録アカウント一覧表示

}

ログイン -> アカウント : アカウントをクリック

}

トップページ -down-> 記事
トップページ -left-> マイページ
トップページ -> 投稿 : 投稿をクリック


[*] --> 管理者
管理者:・記事削除
管理者:・ユーザー削除
```
