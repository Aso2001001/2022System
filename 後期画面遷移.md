```uml
@startuml
[*] --> トップページ

state 記事{
記事一覧 -left-> 記事詳細 : 記事をクリック


記事一覧:・ランダムで表示
記事一覧:・新着順で表示
記事一覧:・カテゴリ別で表示
記事一覧:・観覧数ランキングで表示
記事一覧:・お気に入り登録者数で表示
記事一覧:・特定アカウントで表示
記事一覧:・記事読了後にランダムで表示

}

state マイページ{
ログイン -> アカウント : アカウントをクリック
アカウント -down-> 特殊 : 特殊をクリック
特殊 -down-> 登録アカウント :お気に入りをクリック
特殊 -down-> 特殊記事 :記事をクリック


アカウント:・お気に入り登録者数ランキングで表示

特殊記事:・観覧記事の履歴一覧表示
特殊記事:・お気に入り記事一覧表示
登録アカウント:・お気に入り登録アカウント一覧表示
特殊記事:・お気に入り登録アカウントの投稿記事一覧表示

}


トップページ -down-> 記事
トップページ -left-> マイページ
トップページ -> 投稿 : 投稿をクリック
投稿:・記事を投稿
投稿:・コメントを投稿
投稿:・スタンプを送る
投稿:・動画を投稿
投稿:・時間指定して投稿
投稿:・記事の公開非公開の設定
投稿:・サムネイルを設定



[*] --> 管理者
管理者:・記事削除
管理者:・ユーザー削除


@enduml
```
